# GLayout

HTML5 AND CSS3 SUPPORT REQUIRED

BEST WORKS AT 1080P IN LANDSCAPE MODE
Tested only on Microsoft Edge(Windows 10 Home 64-bit 1607) and Internet Explorer 11.0.14393.0
